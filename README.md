Installation / Usage
--------------------

To install use pip:

    $ pip install ngdlm

    $ pip install git+https://github.com/ai-guru/ngdlm.git


Or clone the repo:

    $ git clone https://github.com/ai-guru/ngdlm.git
    $ python setup.py install
